  ### Commands

  * `mintlify-scrape section [url]` - Scrapes multiple pages in a site.

  * `mintlify-scrape page [url]` - Scrapes a single page in a site.

  ### One-time use:

  <CodeGroup>
    ```typescript Section
    npx @mintlify/scraping@latest section [url]
    ```

    ```typescript Page
    npx @mintlify/scraping@latest page [url]
    ```
  </CodeGroup>
