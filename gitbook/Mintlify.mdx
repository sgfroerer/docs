  ### Commands

  * `mintlify-scrape section [url]` - Scrapes multiple pages in a site.

  * `mintlify-scrape page [url]` - Scrapes a single page in a site.

  ### One-time use:

  <CodeGroup>
    ```typescript Section
    npx @mintlify/scraping@latest section [url]
    ```

    ```typescript Page
    npx @mintlify/scraping@latest page [url]
    ```

<Accordion title="JS & CSS Libraries" defaultOpen={false}>
  To add custom JavaScript (JS) and CSS libraries to a Mintlify project synced with GitHub, you can follow these steps:

  ### Use External Libraries

  <Tip>
    If you're adding an external library (like jQuery, Tailwind, or Chart.js), include their CDN links
  </Tip>

  ```markdown
  <!-- In the <head> -->
  <link href="https://cdn.jsdelivr.net/npm/library-name@version/library.min.css" rel="stylesheet">

  <!-- Before </body> -->
  <script src="https://cdn.jsdelivr.net/npm/library-name@version/library.min.js"></script>
  ```

  <Accordion title="Custom Components/Scripts" defaultOpen={false}>
    <Tip>
      If you're using React (Mintlify's default framework), you can import libraries directly in your React components
    </Tip>

    <Tabs>
      <Tab title="Install">
        *Install the library via npm*

        ```
        npm install library-name
        ```
      </Tab>

      <Tab title="Import">
        *Import it in your React files*

        ```markdown
        import 'library-name/dist/library.css';
        import LibraryName from 'library-name';
        ```
      </Tab>

      <Tab title="Sync">
        *Sync Changes to GitHub*

        ```bash
        git add .
        git commit -m "Add custom JS/CSS files"
        git push
        ```
      </Tab>
    </Tabs>
  </Accordion>
</Accordion>

<Accordion title="Adding External CSS & JavaScript Libraries" defaultOpen={false}>
  **Add the CSS Library to `headLinks`:**

  <Tip>
    In your `mint.json`, include the `link` tag as an object in the `headLinks` array
  </Tip>

  ```json
  "headLinks": [
      {
          "rel": "stylesheet",
          "href": "https://cdn.jsdelivr.net/npm/library-name@version/library.min.css"
      }
  ]
  ```

  **Add the JavaScript Library to `headScripts`:**

  <Tip>
    In your `mint.json`, include the `script` tag as an object in the `headScripts` array
  </Tip>

  ```json
  "headScripts": [
      {
          "src": "https://cdn.jsdelivr.net/npm/library-name@version/library.min.js",
          "async": false
      }
  ]
  ```

  The `async` property can be set to `true` or `false` depending on whether you want the script to load asynchronously

  <Tabs>
    <Tab title="CSS">
      CSS libraries should typically go in `headLinks` since they are loaded in the `<head>`
    </Tab>

    <Tab title="JavaScript">
      JavaScript libraries should be added to `headScripts`, which places them near the end of the body, just before `</body>`
    </Tab>
  </Tabs>
</Accordion>
